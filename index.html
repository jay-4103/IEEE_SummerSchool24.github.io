Web Fundamentals

Master the fundamentals of web development with HTML, CSS, and JavaScript. This bootcamp will cover the core technologies needed to build the foundation of any web application.

Participants will learn how to structure content with HTML, style with CSS, and add interactivity with JavaScript, providing a solid base for further web development skills.

Advanced Frameworks

Explore advanced frameworks like React and Firebase to build dynamic, responsive web applications. Learn how to create modern user interfaces and integrate with backend services.

This section of the bootcamp will introduce you to React for building interactive UIs and Firebase for easy backend integration, allowing you to create full-stack web applications in just two days.
